## Dart im Kontrast zu Java

#### Das "new" Schlüsselwort kann in Dart weggelassen werden
Foo foo = Foo();

### Variablen deklarieren

#### Java
String foo = "abc";

#### Dart
 
var/final foo = "abc";  
String foo = "abc"; 

### \[] Syntax
#### Java
Ein Array wird erstellt.

#### Dart
Eine Liste wird erstellt.
